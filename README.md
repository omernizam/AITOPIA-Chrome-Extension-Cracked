## üì¶ Installation

### Manual Installation (Developer Mode)

This extension is currently available as a developer build and needs to be installed manually. Follow these steps:

**Step 1: Download the Extension**
1. Download or clone this repository to your local machine
2. Extract all files to a folder on your computer

**Step 2: Enable Developer Mode in Chrome**
1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Toggle "Developer mode" ON in the top right corner

**Step 3: Load the Extension**
1. Click the "Load unpacked" button
2. Select the folder containing the extension files
3. The extension should now appear in your extensions list
4. Pin the extension to your toolbar for easy access

**Video Tutorial**
üìπ For a detailed walkthrough, watch this installation guide: [How to Install Chrome Extension Manually](https://www.youtube.com/watch?v=yVvvA8kaIuk)

**Note**: Since this is a developer extension, Chrome may show warnings about unrecognized extensions. This is normal for manually installed extensions.

## üîë API Configuration

To use premium AI models, you'll need to configure API keys:

1. Click the **Settings** gear icon in the extension
2. Navigate to **API Keys** section
3. Add your API keys for the services you want to use:
   - OpenAI API Key (for GPT models)
   - Anthropic API Key (for Claude)
   - Google AI API Key (for Gemini)
   - DeepSeek API Key

### Getting API Keys
- **OpenAI**: [platform.openai.com](https://platform.openai.com/api-keys)
- **Anthropic**: [console.anthropic.com](https://console.anthropic.com/)
- **Google AI**: [makersuite.google.com](https://makersuite.google.com/app/apikey)
- **DeepSeek**: [platform.deepseek.com](https://platform.deepseek.com/)

## üÜò Troubleshooting

### Common Issues

**Extension not loading?**
- Make sure Developer mode is enabled in `chrome://extensions/`
- Try reloading the extension by clicking the refresh button
- Check that all extension files are in the correct folder structure
- Restart Chrome and try loading the extension again

**API errors?**
- Verify your API keys are correct and have sufficient credits
- Check your internet connection
- Some models may have usage limits

**Slow responses?**
- Try switching to a different AI model
- Check if your API provider is experiencing downtime

### Getting Help
- Check the installation video tutorial: [YouTube Guide](https://www.youtube.com/watch?v=yVvvA8kaIuk)
- Review Chrome's official documentation on loading unpacked extensions
- Make sure you're using a supported Chrome version (Chrome 88+)
